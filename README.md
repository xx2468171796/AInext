# Alone模式 - AI持久输出助手

免费开源版 VSCode 扩展 - AI结束前自动弹窗确认，让AI持续工作直到你满意。

## 功能特性

- AI 输出结束前自动弹窗确认
- 让 AI 持续工作直到你满意为止
- 简单易用的控制面板
- 支持自定义提示音

## 安装

1. 在 VSCode 扩展市场搜索 "Alone模式"
2. 点击安装

## 使用方法

1. 安装扩展后，侧边栏会出现 "Alone模式" 图标
2. 点击打开控制面板
3. 开启 Alone 模式后，AI 在结束输出前会弹窗询问是否继续
